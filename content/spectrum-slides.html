Agentic coding 的四個層級

Level 1：Vibe coding

  • 讓 AI 自己去搞
  • 當作程式碼不存在
  • 錯誤 log？直接丟給 AI
  • 版本控制？那是什麼？

做一次性的專案沒問題

別指望它能長期維護

Level 2：有紀律的 agentic coding

  • 版本控制 - Git 和 GitHub
  • 檔案層級的理解
  • 基本的安全防護
  • 測試 - 確認東西真的能動

Level 3：Agentic 軟體工程

  • 你有真的去驗證的測試
  • Pre-commit hook
  • CI job 做持續測試
  • 函式層級的理解

額外的品質控管措施

  • 仔細的手動測試
  • End-to-end 測試
  • 一次性的 AI code review
  • Regression 檢查

Level 4：高品質軟體工程

  • 逐行理解程式碼
  • 自我反思迴圈
  • 互動式 AI code review
